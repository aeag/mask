Mask

Purpose

Help to create mask...

Principle

When one or more objects are selected, activating the plugin generates a polygon memory layer containing a "mask" whose dimension is twice area of the current view.

The mask is a gray box, with holes in order to highlight the underlying objects

For advanced users, you can trigger the creation of masks from the python console

# import plugin
from mask import aeag_mask
# Classic call : mask the selected features
aeag_mask.do()
# advanced call : custom geometry sr = QgsCoordinateReferenceSystem() sr.createFromSrid(2154) geom = QgsGeometry.fromWkt("POLYGON((356419 6120047, 1051423 6120047, 1051423 6595985, 356419 6595985, 356419 6120047))") aeag_mask.do(sr, { geom }, "MyMask")
